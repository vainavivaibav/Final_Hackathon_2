# AI Supply Chain Intelligence Dashboard

An AI-powered logistics and supply chain optimization platform built using FastAPI, Streamlit, and Machine Learning (gradient boosting models).

This project predicts demand, detects delays, optimizes suppliers, and provides real-time decision support for logistics operations.

---

## Features

- Demand Forecasting (7-day prediction)
- Delay Risk Prediction (ML-based)
- Inventory Optimization (Reorder Point)
- Supplier Selection & Optimization
- Route Intelligence & Distance Calculation
- Sustainability Tracking (CO₂ Emissions)
- Resilience Analysis (Disruption Scenarios)
- AI Decision Engine
- Interactive Dashboard (Streamlit)

---

## Tech Stack

- Frontend: Streamlit
- Backend: FastAPI
- Machine Learning:
  - HistGradientBoosting (XGBoost-style)
  - Gradient Boosting (Ensemble)
- Libraries: pandas, numpy, scikit-learn, plotly
